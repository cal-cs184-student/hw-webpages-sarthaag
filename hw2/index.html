Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

de Casteljau's algorithm begins with choosing an arbitrary number of points in a plane and connecting them together. Then, points are inserted on each edge through linear interpolation and those are connected together. This process repeats recursively until a final point is found that defines the Bezier curve. To implement this, we implemented functions that did one step of this algorithm in 2D and 3D, using for loops to iterate over all pairs of adjacent points and compute the linearly interpolated point between each pair. Then the last function recursively calls "evaluateStep" on the input points until only one is left, which is the one we're looking for.

Part 2: Bezier surfaces with separable 1D de Casteljau

de Casteljau algorithms extend to Bezier surfaces because the process is the same recursive interpolation used in Bezier curves but extended to two dimensions. For any point on the surface, the algorithm performs the 1D evaluation along one direction of the grid with parameter u, then again in the orthogonal direction with parameter v. In our implementation, we first loop through each row of control points and call evaluate1D with parameter u to compute an intermediate point for each row. These intermediate points are stored in a temporary vector. We then call evaluate1D again on this vector with parameter v, which evaluates the final Bézier curve and returns the point on the Bézier surface at (u, v).

Section II: Triangle Meshes and Half-Edge Data Structure

Part 3: Area-weighted vertex normals

Starting from the vertex’s half-edge, we get the coordinates of the three vertices of the corresponding triangle. We then compute two edge vectors from these points and compute the face normal by taking the cross product of these vectors. Since the magnitude of the cross product is proportional to twice the triangle’s area, each cross product weights each face normal by its area. After summing the normals for all faces by using twin()->next() to get to the next half-edge, we normalize the resulting vector to produce the unit vertex normal.
